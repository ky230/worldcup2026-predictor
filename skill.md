## 二、数据与舆情预测决策环路 (Prediction Loop Workflow)

> **说明**：每次生成新比赛日 HTML 时，**按照以下三个阶段顺序执行**，不可跳过或合并阶段。Loop 目标是把专业数据与场外情报全部榨干后，再输出 HTML 面板。详细的数据源链接、搜索Query及标注规范，请统一参阅 [data_sources.md](file:///Users/ky230/Desktop/FF2026/reference/data_sources.md)。

### 【阶段一】专业数据深挖 · 锚定基础实力

**执行原则**：不只看阵容本身，重点分析「形势层」。对每场比赛，依次查阅以下具体数据页面与指标：
1. **FotMob 核心指标与页面查询**：对对阵双方在数据手册中查找具体的 FotMob URL：
   - **赛程与预计首发 (🚨 最关键的开局首步)**：打开 [data_sources.md](file:///Users/ky230/Desktop/FF2026/reference/data_sources.md) 指引的 Fixtures by Date 找到对应 Match Details 页面。**这是整个 Loop 最核心的第一步，Agent 必须通过此页面彻底搞清楚当前在预测哪两个队、开球时间，以及可能的预计首发阵容与伤停**。核对首发阵容、伤停名单、近 5 场两队球员评分走势。**在此页面中的 H2H (Head-to-head) 栏目中检索两队的历史交锋战绩（🚨 历史交锋限制：必须过滤掉年份超过 5 年的交手记录，只参考最近 5 年之内的对决，超过 5 年的记录没有参考价值）**。**生成的 HTML 面板中的首发阵型、球员姓名与位置，必须与 FotMob 网页上所看到的完全保持一致，做到“网站所见即面板所见”，绝对不允许出现名字错位或颠倒**。
   - **球员多维数据**：打开 [data_sources.md](file:///Users/ky230/Desktop/FF2026/reference/data_sources.md) 指引的 [球员统计数据](https://www.fotmob.com/leagues/77/stats/world-cup/players) 及其子项直达链接（包含进球 Goals、助攻 Assists、期望进球 xG、期望助攻 xA、创造重大机会 Big chances created 与 FotMob 评分），核对两队核心球员的个人数据，以定性分析球星战力。
   - **团队攻防战术**：打开 [data_sources.md](file:///Users/ky230/Desktop/FF2026/reference/data_sources.md) 指引的 [球队统计数据](https://www.fotmob.com/leagues/77/stats/world-cup/teams)。核对两队的 Expected Goals (xG) 与 Expected Goals Conceded (xGA)（即 Defense 分类下的 xG conceded）；查看 Average possession 和 Accurate passes per match 判定谁掌控中场；使用 **Possession won final 3rd per match**（在 Defense 分类中）作为逼抢强度的直接代理指标（因 FotMob 无原生 PPDA）。
2. **裁判与比赛环境核对**：直接在上述第一步的 **FotMob 单场 Match Details 页面**（Match facts / Overview 模块）中查阅核对：
   - **主裁判信息**：核对裁判指派安排，必要时检索其历史执法尺度与给牌率，判定防守型/侵略性战术修正；
   - **球场与环境**：确认比赛球场、城市、实时气温与降雨等。如果为高原城市（如墨西哥城、瓜达拉哈拉等 1500m 以上海拔），或实时气温 ≥ 32°C，必须触发本手册【3-A】中的胜率修正规则。

---

### 【阶段二】场外雷达 · 更衣室舆情侦察

**执行方针**：场外情报是**概率校正因子**。场外情报的价值在于识别「纸面实力与实际发挥之间的折扣」。
1. **分级舆情与伤停核对**：根据 [data_sources.md](file:///Users/ky230/Desktop/FF2026/reference/data_sources.md)【第三部分 3-A】所确立的分级优先级检索舆情和实时伤停：
   - **第一优先级**：优先查阅 **FotMob 单场比赛详情页**（Insights & News 实时伤停/红黄牌更新、Team Form 走势、About the Match 赛事信息）。
   - **第二优先级**：查阅 **Sofascore 新闻专区** 及 **FIFA 官方新闻** 验证世界杯赛事大盘新闻与伤退官宣。
   - **第三优先级**：使用 [data_sources.md](file:///Users/ky230/Desktop/FF2026/reference/data_sources.md)【3-B】所列的 **Google 搜索 Query 模板** 或官方社媒，核查深层舆情（如更衣室内讧细节、战术泄密等）。
2. **多信源校验**：依据 [data_sources.md](file:///Users/ky230/Desktop/FF2026/reference/data_sources.md) 的【3-C 舆情爆料源信源分级与可信度目录】核查新闻可信度。只有当 1 个以上一级媒体，或 2 个以上二级媒体独立证实后，方可启动胜率下调机制。
3. **后勤与环境排查**：直接在 FotMob 单场比赛详情页（About the Match 模块等）检索开球时间，核查当地海拔和气温是否触发本手册【3-A】节的概率扣分规则（无需考虑主客场差旅时区差）。

### 【阶段三】信息融汇与研判决策逻辑

**执行原则**：「用数据定锚，用情报调偏」。先确定基础概率（来自阶段一的硬数据），再用阶段二的情报作为「修正因子」。禁止逆序操作。

#### 3-A. 基础概率锚定公式

```
基础胜率 = 
  近期状态(40%) × 形态分
+ 硬实力底盘(30%) × 世界档位
+ 历史交锋(15%) × 中立场胜率 (🚨 历史交锋限制：仅参考 5 年之内的交手记录；年份超过 5 年的交手记录无参考价值，该项 15% 权重直接归 0 均摊至近期状态 47% 与硬实力 38% 中)
+ 情境因素(15%) × [东道主加成 / 天气/海拔修正 / 淘汰赛抗压值]
```

情境因素中**天气/海拔修正标准值**：
* 气温 ≥ 32°C：高位逼抢体系（PPDA<8）欧洲队，**胜率 -3%~-5%**；低重心防守反击体系，**胜率 +2%~+3%**（受益方）。
* 海拔 ≥ 1500m（如丹佛、墨西哥城）：非本洲球队，**胜率 -4%~-8%**；本洲球队 +4%~+6%。
* 前一天深夜赛后短暂休息（<48h）：任何球队，**体能系数 -3%**。

#### 3-B. 情报修正因子（场外减分/加分系数）

| 情报类型 | 确认门槛 | 概率修正 | 波胆影响 |
|---------|---------|---------|---------|
| **更衣室内讧（确认）** | ≥2个独立一级媒体源 | **热门方 -8%~-15%** | 意外比分权重+5%，放大冷门尾部概率 |
| **更衣室内讧（单源爆料）** | 仅1个媒体源 | **-3%~-5%** | 在舆情哨中标注，波胆不变 |
| **核心球员伤停（官方确认）** | FotMob移出首发 OR 官方声明 | 按球员影响力 **-5%~-20%** | 主力核心级（如梅西/姆巴佩）≥-15% |
| **本届大赛中球队已露出疲软信号** | 首轮实际比赛结果 | 下修信心系数，胜率区间扩散 | 防冷概率+3% |

#### 3-C. 国家/球队内讧体质校准表
以下为历史大赛数据校准的"内讧折扣系数"。当爆料发生时，按此表决定修正力度而非一刀切。
* 🇫🇷 **法国** (内讧体质：🔴 HIGH)：历史多发更衣室事件（2010罢训）。确认内讧 → **-15%**。
* 🇨🇲 **喀麦隆** / 🇳🇬 **尼日利亚** (内讧体质：🔴 HIGH)：常有主帅/球员冲突或权力争夺。确认内讧 → **-15%~-20%**。
* 🇧🇪 **比利时** (内讧体质：🟠 MED-HIGH)：荷语/法语圈派系分化背景音强。确认内讧 → **-10%~-12%**。
* 🇵🇹 **葡萄牙** (内讧体质：🟠 MED)：C罗话语权与球星定位风波。C罗相关更衣室爆料 → **-8%**。
* 🇦🇷 **阿根廷** (内讧体质：🟡 MED)：梅西体系稳定时较低（-3%），若核心有争议则下调 **-10%**。
* 🇳🇱 **荷兰** (内讧体质：🟡 LOW)：历史多内讧但上场独立性强。确认内讧 → **-2%~-3%**。
* 🇩🇪 **德国** / 🏴󠁧󠁢󠁥󠁮󠁧󠁿 **英格兰** (内讧体质：🟡 LOW)：管控较严。确认内讧 → **-5%**。
* 🇯🇵 **日本** / 🇰🇷 **韩国** (内讧体质：🟢 VERY LOW)：集体荣誉文化，基本不修正，仅在舆情哨注明。

#### 3-D. 研判决策流程（当数据与舆情冲突时）
```
Step 1: 确认「热数据」 → 阶段一的 xG、近5场评分、伤停、天气
Step 2: 确认「冷情报」 → 阶段二的内讧/伤停烟雾弹分级
Step 3: 依据硬数据锚定基础概率（Step 1）
Step 4: 叠加情报修正因子进行概率微调（Step 2 × 3-B/3-C校准）
Step 5: 最终胜/平/负概率三数加总=100（允许平局概率，但单支队伍胜率不超过85%）
Step 6: 输出让球与波胆预测结果。让球指数预测必须给出且仅给出 2 个推荐（1 个首选 + 1 个次选）；波胆/比分倾向必须给出且仅给出 4 个推荐（2 个稳胆 + 2 个防冷）。
Step 7: 在「更衣室晴雨表」模块写入当前侦察结论
```

## 四、球队资料库（核心依据）

### 夺冠热门档

**阿根廷**（J组）：梅西（39岁，迈阿密国际，最后一舞）。**战术特色**：**麦卡利斯特与恩佐双核调度默契**，德保罗在中右路护航，劳塔罗（高位拦截为梅西省体能）与阿尔瓦雷斯在单双锋线切换。**隐忧与雷区**：梅西体能不足以支撑全场逼抢；奥塔门迪身后存在反击打穿的真实风险。德保罗停赛时防守强度下降 15%-20%。

**西班牙**（H组）：**战术特色**：**亚马尔（18岁）与尼科·威廉姆斯两翼齐飞**，中场罗德里（定海神针）覆盖面大，佩德里或奥尔莫担任前插攻击手。**隐忧与雷区**：年轻主力大赛逆境经验偏少；亚马尔未经历深度打压考验。中锋莫拉塔心理脆弱，罗德里单防跑动时两侧身后易暴露。

**法国**（I组）：姆巴佩（皇马，队长，巅峰期）、楚阿梅尼、萨利巴、老将坎特（费内巴切）。**战术特色**：**奥利塞与姆巴佩化学反应良好**，目前阵型为「奥利塞（右路内收做半个#10）+ 楚阿梅尼/拉比奥（中场前插）+ 坎特（防守屏障）」的三角体系。**隐忧与雷区**：格列兹曼退后组织缺失，依赖边路推进。中场老将坎特面临体能天花板，且若其停赛/伤退，防守屏障将严重失衡。德尚换人保守。**内讧体质 HIGH（-15%）**。

**英格兰**（L组）：**战术特色**：强调宽度 + 贝林厄姆前腰插上的 4-2-3-1 或 4-3-3 灵活切换。**贝林厄姆与中锋凯恩交错换位**，前插抢点。福登是体系“隐藏枢纽”，福登不在时贝林厄姆消耗过大。**隐忧与雷区**：大赛心魔；马奎尔高位压迫下出球速度慢易被抢断；缺乏可靠的禁区内定位球套路，转化率极低。

**巴西**（C组）：**战术特色**：趋向 4-2-3-1。**维尼修斯与拉菲尼亚两翼内切极具威胁**，罗德里戈穿插及恩德里克冲击。**隐忧与雷区**：维尼修斯在阵地战中盘带易被封死，依赖罗德里戈的创造力。左边后卫接班人稳定性是最大未知数。

**荷兰**（F组）：**战术特色**：**西蒙斯（Xavi Simons）是中场创造核心**，自由人角色与德容搭配。**隐忧与雷区**：缺乏阵地战一锤定音的中锋；德容踝部伤病史是必查项；首场暴露出被高位逼抢时推进滞涩问题。

**乌拉圭**（H组）：贝尔萨执教，巴尔韦德领衔，努涅斯、阿劳霍。强调全场高压逼抢与快速垂直进攻。

**克罗地亚**（L组）：莫德里奇（40岁，谢幕战）、科瓦契奇、格瓦迪奥尔。善打加时与防反，核心老化，逼抢强度下滑。

**摩洛哥**（C组）：阿什拉夫、布拉欣·迪亚斯、恩内斯里。防守硬度与快速转换依然是非洲第一档。

**哥伦比亚**（K组）：**战术特色**：快速边路反击 + 紧凑中场屏障。**路易斯·迪亚斯（左翼）**是真正核心。**隐忧与雷区**：J罗的更衣室关系起伏不稳；迪亚斯伤病史需重点观察；中后防线在高压逼抢下易失误。

**日本**（F组）：**战术特色**：三笘薰在左路 1v1 爆破，久保建英在右路精细出球。**佐野海舟顶替伤退的远藤航，与守田英正搭档双后腰，虽硬度下降，但运动能力和出球速度更快**。**隐忧与雷区**：缺乏强力中锋，高空球防守易被欧洲/非洲强队针对。

**挪威**（I组）：哈兰德与厄德高“厄哈连线”是世界级杀器。隐忧：大赛零经验，后防防空与防快攻能力是短板。

## 五、红线（必须遵守）

1. 不贬损任何国家、球队、球员。
2. 不编造具体比分历史、虚构伤病信息。
